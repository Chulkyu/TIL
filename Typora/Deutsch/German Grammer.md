### Lektion 8. 소유관사

```
소유관사 격 :  그의 아내는 -> 1격
						그의 아내에게 -> 3격
						그의 아내를 -> 4격
						
소유관사 성 : 그의 아내는 -> 여성
					 그의 아버지는 -> 남성
					 그의 아이는 -> 중성
					 그의 아이들은 -> 복수
```

```
내 아버지는 선생님입니다.
-> Mein Vater ist Lehrer
네 어머니도 선생님입니다.
-> Deine Mutter ist auch Lehrerin

그의 남동생은 의사입니다.
-> Sein Bruder ist Arzt
그녀의 여동생도 의사입니다.
-> Ihre Schwester ist auch Ärztin

그 아이는 책을 가지고 있다.
-> Das Kind hat ein Buch
그(녀)의 책은 작다.
-> Sein Buch ist klein

우리의 차는 파랗다.
-> Unser Auto ist blau
너희의 차들은 노랗다.
-> Eure Autos sind gelb

그녀(그들, 당신)의 책들은 흥미롭다.
-> Ihre Bücher sind interessant

이 자동차는 내 어머니 마음에 든다.
-> Das Auto gefällt meiner Mutter
이 집은 너의 아버지 마음에 든다.
-> Das Haus gefällt deinem Vater

한 남자가 그의 아내를 돕는다.
-> Ein Mann hilft seiner Frau
한 여자가 그녀의 남편을 돕는다.
-> Eine Frau hilft ihrem Mann

그 가방은 우리 어머니 마음에 든다.
-> Die Tasche gefällt unserer Mutter
그건 너희 여동생 마음에도 든다.
-> Sie gefällt auch eurer Schwester
그 자동차는 내 부인 거다.
-> Das Auto gehört meiner Frau

나는 나의 아버지를 부른다.
-> Ich rufe meinen Vater

한 여자가 그녀의 형제에게 질문했다.
-> Eine Frau fragt ihren Bruder
한 남자가 그의 형제들에게 질문했다.
-> Ein Mann fragt seine Brüder

우리는 우리의 집을 본다.
-> Wir sehen unser Haus
너희는 너희의 집들을 본다.
-> Ihr seht eure Häuser

그 여선생님은 그녀의 학생들에게 질문한다.
-> Die Lehrerin fragt ihre Schüler
그 여선생님은 당신의 학생들에게 질문한다.
-> Die Lehrerin fragt Ihre Schüler

+ 소유관사 2격(주어 뒤에서 꾸며준다.)
내 아이의 친구들이 자신의 선생님에게 질문한다.
-> Die Freunde meines Kindes fragen ihren Lehrer


```

```

```
1. Verb + 1격

Es regnet : 비가 온다.
Ich singe : 난 노래한다.
Es hat geregnet : 비가 왔다.
Ich sang : 난 노래 했다.
```

```
2. Verb + 1격 + 1격

Ich bin Lehrer : 난 선생님이다.
Meine Tochter ist Schülerin : 내 딸은 학생이다.
Ich wurde Lehrer : 난 선생님이 되었다.
Sie ist Studentin geworden : 그녀는 여대생이 되었다.
```

```
3. Verb + 1격 + 4격

너는 날 사랑해? : Liebst du mich?
나는 그를 방문한다. : Ich besuche ihn.
난 숙제를 했다. : Ich habe Hausaufgabe gemacht.
너는 나를 방문해줄 수 있어? : Kannst du mich besuchen?
드디어 방 하나를 찾았습니다. : Endlich habe ich ein Zimmer gefunden.

```

```
4. Verb + 1격 + 3격
	: danken, folgen, gefallen, gehören, glauben,
	  gratulieren, helfen, schmecken.

초대해 줘서 고마워. : Ich danke dir für die Einladung.
저 책은 내 거다. : Das Buch gehört mir.
파티는 내 맘에 들었다. : Die Party hat mir gut gefallen.
저 여인이 그들을 도왔다. : Die Frau hat ihnen geholfen.

```

```
5. Verb + 1격 + 전치사격 보충어
  : sich + 동사원형 + 전치사 (숙어)
	: sich freuen auf + 4격, warten auf + 4격, denken an + 4격
	
널 이미 한 시간 동안 기다리고 있다.
 : Auf dich warte ich schon eine Stunde lang.
난 항상 그녀를 생각하고 있다.
 : Ich denke immer an sie.
내 부모님은 내 선물에 매우 기뻐하셨다.
 : Meine Eltern haben sich sehr über mein Geschenk gefreut.

```

```
6. Verb + 1격 + 3격 + 4격
	: empfehlen, geben, erzählen, schenken, zeigen ...

 : Würden Sie mir bitte die Speisekarte bringen?
 : Würden Sie sie mir bitte bringen?

 : Mein Nachbar hat mir einen Kuli geschenkt.
 : Mein Nachbar hat ihn mir geschenkt.
나는 너희들에게 휴가 사진을 보여준다.
 : Ich zeige euch die Urlaubsfotos.
 : Ich zeige sie euch.
당신에게 좋은 주말을 기원 드려요.
 : Ich wünsche Ihnen ein schönes Wochenende.
