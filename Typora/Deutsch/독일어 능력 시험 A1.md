### Hoerverstehen 1.

```
example.
F : Ach, Verzeihung, wo finde ich Herrn Schneider vom Betriebsrat?
M : Schneider. Warten Sie mal. Ich glaube, er ist in Zimmer Nummer 254.
	  ja, stimmt, Zimmer 254. Das ist im zweiten Stock.
	  Da können Sie den Aufzug dort nehmen.
F : Zweiter Stock, Zimmer 254. Okay, vielen Dank.

die Verzeihung :  실례, 용서
finden : 발견하다
vom Betriebsrat : 노동조합의
warten : 기다리다
glauben : 생각하다, 믿다
das Zimmer : 방
der Aufzug : 엘리베이터
nehmen : 타다, 잡다
```

```
1.
M : Hi Albert! Wo wollen wir diesen sommer Urlaub machen?
A : Wie findest du eine Stadtreise? vielleicht nach London?
M : London ist toll, aber ich war schon zweimal dort.
		Wir können auch nach Paris fahren.
A : Paris ist auch toll oder auch Wien?
M : Ich will lieber nach Paris. Da war ich noch nie.
A : Gut, dann machen wir es so.

der Sommer : 여름
der Urlaub : 휴가
wollen : ~을 원하다 (화법조동사)
die Stadtreise : 도시 여행
vielleicht : 어쩌면
toll : 멋진
schon : 이미
zweimal : 두 번
dort : 거기에서
können : ~할 수 있다(화법조동사)
***lieber : 차라리~(gern 비교급)
```

```
2.
M : Sag mal, Lena. Kommst du zur Party von Joseph?
L : Ja, natürlich. Bringst du was mit?
M : Ja, ich bringe einein Salat mit. Und du? Was nimmst du mit?
L : Ich weiß nicht. Soll ich ein paar CDs mitbringen?
M : Ich denke, die hat Joseph schon vorbereitet.
		Wie wäre es mit einem Wein?
L : Ein Wein? Rot order Weiß?
M : Wie du willst.
L : Okay, dann bringe ich einen Rotwein mit.

Zur Party : 파티로
natürlich : 당연히
mitbringen : 가지고 오다
mitnehmen : 가지고 가다
der Salat : 샐러드 
haben ... vorbereitet : 준비했다 (vorbereiten의 현재완료)
Wie wäre es mit...? : ...은 어때?
Wie du willst : 너가 원하는대로.
```

```
3.
F : Enschuldigen Sie bitte.
M : Ja bitte.
F : Können Sie mir sagen, wie spät es ist?
M : Ja, es ist halb 4.
F : Oh, ich habe keine Zeit. Wo ist das Gleis 3?
M : Gehen Sie geradeaus und biegen Sie dann rechts ab.
F : Vielen Dank.

abbiegen : 방향을 바꾸다.
der Ausgang : 출구, 공항게이트
geradeaus : 직진
rechts : 오른쪽
links : 왼쪽
```

```
4.
L : Guten Tag. Können Sie mir helfen? Ich besuche ab heute den A2 Kurs.
		Können Sie mir sagen, wo ich hin muss? Ist Zimmer 311 richtig?
I : Wie ist Ihr Name?
L : Mein Name ist Luisa Wagner.
I : Einen Moment. Sie haben die Prüfung nicht bestanden.
		Deswegen müssen Sie diese wiederholen. Gehen Sie in den Raum 211.
L : Oh, je!

sollen : ~하는 것이 좋다(화법조동사)
haben ... bestanden : 합격했다 (bestehen의 현재완료)
die Prüfung : 시험
wiederholen : 반복하다, 되풀이하다
besuchen + 학습기관 : 학습기관에 다니다.
hin : ~로
deswegen : 그래서
oh, je : 오, 이런 !
```
