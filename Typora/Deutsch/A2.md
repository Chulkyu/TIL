### Lektion 5. 화법조동사

```
- 영어의 조동사
- 동사의 의미 확장을 도움
- 동사원형과 결합 ( * 동사원형 문장 후치 * )
- 1인칭, 3인칭 변화가 똑같다.
```

```
화법조동사 과거
- 화법조동사 과거형에는 움라우트가 사라진다.
- 현재와 대비되는 뜻을 가진다.
 : Ich will Lehrer werden. (선생님이 되고 싶다.)
 -> Ich wollte Lehrer werden.
  (선생님이 되고 싶었다.) '지금은 아니다'
```

```
화법조동사의 접속법2식 ( 간접화법 )

- 공손, 추측.
- ~일 것이다. ~인 것이다. ~일지도 모른다. 
- 조동사의 일반과거형에 움라우트만 붙인다.
- wollen, sollen의 접속법2식은 없다.

```

```
접속법2식 : könnten

- 영어의 could 와 같은 용법 ( 공손 / 추측 )
- können 보다 더욱 공손

나는 피아노를 칠 수 있다.
 : Ich kann Klavier spielen.
과거형   : Ich konnte Klavier spielen.
간접화볍  : Ich könnte Klavier spielen.

저를 도와주실 수 있습니까?
 : Können Sie bitte mir helfen?
간접화법 : Könnten Sie bitte mir helfen?

나를 기다려줄 수 있겠니?
 : Könntest du bitte ** auf mich warten? **

```

```
dürfen : 허가

너는 지금 집에 가도 좋다.
 : Du darfst jetzt nach Hause.
 
여기에서 담배를 피우면 안 된다.
 : Man darf hier nicht rauchen.
```

```
mögen : 소망 ( 좋아하다 )

나는 그를 좋아하지 않는다.
 : Ich mag ihn nicht.

그는 독일에서 공부하고 싶어 한다.
 : Er möchte in Deutschland studieren.

```

```
müssen : 의무, 강한 추측

나는 지금 집에 가야 한다.
 : Ich muss jetzt nach Hause.

핸드폰은 소파 위에 있을 것이 분명하다.
 : Das Handy muss auf dem Sofa sein.

나는 오늘 학교에 안 가도 된다.
 : Ich muss heute nicht zur Schule.

```

```
wollen : 의지
- 정해진 미래에 대한 서술은 아님.

나는 아이스크림을 원해.
 : Ich will ein Eis.
 
우리 금요일에 영화 보러 갈까?
 : Wollen wir am Freitag ins Kino gehen?
 
```

```
sollen : 타자의 의지가 개입된 의무, 충고

그녀는 나에게 일찍 일어나야 한다고 말했다.
 : Sie hat gesagt, ich soll früh aufstehen.

그는 외국에서 죽었다고 한다.
 : Er soll im Ausland gestorben sein.

```

```

### Lektion 6. 분리 / 비분리 동사

```
분리동사

- 분리전철 : 전치사, 부사, 동사
- 현재, 과거 및 명령형에서 전철은 분리하고 문장 뒤에 온다.

- 과거분사에서 ge는 전철과 어간 사이에 위치한다.
  aufstehen -> aufgestanden
- 분리 동사에 zu부정형을 만들 때에는 전철과 어간 사이에 zu를 넣는다.
  aufstehen -> aufzustehen : 기상하는 것.
```

```
자주 쓰이는 분리전철

aufmachen : Kannst du bitte die Tür aufmachen?

an : 닿을 착

anfangen : Fängst du bitte zuerst an?

abnehmen : 떼어내다, 감소하다

ausgehen : 외출하다
				 : Ich gehe heute Abend aus.
ausmachen : Machen Sie bitte den Computer aus.

mit : 함께
		: mitkommen 함께오다, 동행하다
		
nach : ~를 따라서 ( 추적의 의미 )
		 : nachkommen ~을 따라서
		 
bei : ~의 곁에서
		: beistehen 동조하다, 협력하다

		: vorhaben 계획하다

	  : zumachen 닫다
