# 원어민 꿀패턴

------

### Lektion 1. Lass uns + 목적어 + R / Wollen wir + 목적어 + R

```
Teil 1. Lass uns + 목적어 + R

밥 먹으러 가자 : Lass uns essen gehen.
부엌을 청소하자 : Lass uns die Küche aufräumen.
그것에 대해 이야기하자 : Lass uns darüber(drüber) reden.
베를린으로 (놀러) 가자 : Lass uns nach Berlin fahren.

```

```
Teil 2. Wollen wir + 목적어 + R ?

우리 놀러 갈까? : Wollen wir etwas unternehmen?
우리 한번 만날까요? : Wollen wir uns mal treffen?
박물관 가볼까? : Wollen wir ins Museum gehen?
커피 한 잔 하러 갈래요? : Wollen wir einen Kaffe trinken gehen?

입구에서 만나자 : Lass uns am Eingang treffen
게임 한판 하자 : Lass uns ein Spiel spielen
같이 요리 할래? : Wollen wir mal gemeinsam kochen?
영화 보러 갈래? : Wollen wir ins Kino gehen?

무슨 일이야? : Was ist los?

```

### Lektion 2. Bist du fertig mit + (대)명사 (3격)? / Sag mir, wenn du 형 + 동.

```
Teil 1. Bist du fertig mit + 3 ?

다 먹었어? : Bist du fertig mit dem Essen?
졸업했어? : Bist du fertig mit dem Studium?
학교 끝났어? : Bist du fertig mit der Schule?
일 다 했어? : Bist du fertig mit der Arbeit?
걔랑 끝났어? : Bist du fertig mit ihm?

```

```
Teil 2. Sag mir, wenn du 형 + 동

끝나면 말해줘 : Sag mir, wenn du fertig bist.
준비되면 말해줘 : Sag mir, wenn du bereit bist.
일어나면 말해줘 : Sag mir, wenn du aufstehst.
기분 좀 나아지면 말해줘 : Sag mir, wenn du dich besser fühlst.

```

```

1) 공부 다 했어? : Bist du fertig mit dem Lernen?
2) 빨래 다 끝났어? : Bist du fertig mit der Wäsche?
3) 마치면 말해줘 : Sag mir, wenn du Schluss hast.
4) 시간 있으면 말해줘 : Sag mir, wenn du Zeit hast.

좋은 하루 되세요 : Einen schönen Tag noch.
당신도요 : Ihnen auch

```
